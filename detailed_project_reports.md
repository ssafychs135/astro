### 🛠 기술적 구현 상세

    - **발표자 권한 위임 (`host-delegate`)**: 협업 중 호스트 권한을 동적으로 넘겨주는 로직 구현.

## 🍷 2. Jupasu (S14P21A505)
**"고도화된 취향 분석 및 벡터 검색 기반 와인 큐레이션 플랫폼"**

### 🎯 프로젝트 개요
사용자의 설문과 리뷰 이력을 수학적으로 분석하고, 이미지 인식과 벡터 검색을 통해 가장 개인화된 와인 경험을 제공하는 지능형 서비스입니다.

### 🧠 핵심 비즈니스 로직 (Workflow)
1. **베이지안 취향 교정**: 사용자의 설문 데이터(`Stated`)와 실제 리뷰 이력(`Revealed`)을 베이지안 모델로 혼합합니다. 최신 리뷰에 더 높은 가중치를 주는 **시간 감쇠(Time Decay)** 알고리즘을 통해 실시간 취향을 정밀하게 도출합니다.
2. **리스크 회피형 그룹 추천**: 여러 명의 취향을 결합할 때, 한 명이라도 특정 맛을 '극불호(1점)'로 표시하면 해당 속성의 수치를 강제로 낮추는 **리스크 회피 알고리즘**을 적용하여 모두가 만족하는 와인을 추천합니다.
3. **점수 최적화 (Sigmoid)**: 계산된 복잡한 매칭 점수를 **Sigmoid 함수**에 통과시켜, 사용자에게는 70~95% 사이의 직관적이고 신뢰감 있는 백분율 점수로 변환하여 제공합니다.
4. **벡터 유사도 검색**: `pgvector`의 코사인 유사도(`<+>`) 연산을 활용해 수만 개의 와인 중 취향에 가장 근접한 리스트를 도출합니다.

### 🛠 기술적 구현 상세
- **지능형 스캔 파이프라인**: 
    - **신뢰도 필터링**: OCR 결과 중 임계값(Threshold) 이하의 노이즈 데이터를 선제 제거.
    - **LLM 컨텍스트 정제**: 파편화된 단어를 LLM(FastAPI)에 전달하여 와인 도메인 정보로 재구성.
    - **고정밀 퍼지 검색**: `pg_trgm` 기반의 `word_similarity` 연산으로 최종 매칭 정확도 극대화.
- **아키텍처**: `Java 21` 기반 비즈니스 로직과 `FastAPI` 기반 AI 연산 서버의 하이브리드 구성.

## 🔍 3. Team9 Project (team9)
**"전문가 페르소나 기반 리뷰 분석 및 미디어 요약 서비스"**

### 🎯 프로젝트 개요
산재된 쇼핑몰 리뷰와 유튜브 동영상의 자막을 AI로 정제하여, 사용자가 원하는 전문적인 인사이트를 빠르게 제공하는 분석 플랫폼입니다.

### 🧠 핵심 비즈니스 로직 (Workflow)
1. **전문가 페르소나 요약**: 유튜브 자막을 분석할 때 **'IT 제품 리뷰 전문가' 페르소나**를 주입하여 제품의 장단점을 구조적으로 분석한 전문 요약본을 생성합니다. (모델: `gemini-2.5-flash-lite`)
2. **Self-Querying RAG**: 사용자의 자연어 질문을 AI가 분석하여 모델명, 출시일(`released_date`), 출처(`source`) 등의 **메타데이터 필터**를 스스로 생성하고 `ChromaDB`에서 정확한 근거 데이터를 찾아 답변합니다.
3. **지능형 라우팅**: 질문의 의도를 분석하여 단순 대화와 정밀 데이터 검색(RAG) 체인을 분리 실행함으로써 비용과 속도를 최적화합니다.

### 🛠 기술적 구현 상세
- **비대칭 워커 아키텍처**: `Django`는 비즈니스 로직에 집중하고, `Node.js` 워커는 유튜브 자막 수집 및 Gemini API 연동과 같은 대용량 비동기 I/O를 전담 처리.
- **정교한 데이터 수집**: `Selenium`과 `WebDriverManager`를 통해 동적 렌더링 페이지의 리뷰 데이터를 안정적으로 크롤링.
- **한국어 최적화**: `Upstage Solar` 모델을 통해 국내 리뷰어들의 미묘한 감성 표현을 정확히 식별.
